# Astro3D.ru — Астрологический процессор онлайн 2D и 3D

Интерактивный веб-инструмент для профессиональной и любительской астрологии.

Проект переведен на высокопроизводительную гибридную архитектуру:
1. **Клиентская часть (Фронтенд):** Чистый адаптивный интерфейс (HTML5, CSS3, ES-модули JavaScript), визуализация 2D-карт, интерактивные финансово-астрологические графики на Plotly.js и трехмерный рендеринг чакральной натальной карты на Three.js (WebGL).
2. **Серверная часть (Бэкенд Go API):** Высокопроизводительный сервер на языке Go, выполняющий роль полноценного астрологического ядра. Интегрирован напрямую с математическим пакетом расчетов, исключая медленные консольные вызовы CLI (`os/exec`). Включает автоматическое In-Memory кэширование расчетов, CORS, расширенную фильтрацию и встроенную динамическую поддержку часовых поясов.

---

## Ключевые изменения архитектуры бэкенда

В рамках масштабной оптимизации архитектуры сервера были реализованы следующие изменения:

1. **Прямая интеграция вычислений (Устранение `os/exec`):**
   - Убрана прослойка запуска внешних CLI-процессов.
   - Вычисления выполняются напрямую в памяти Go-приложения путем импорта пакетов `astro` и `interactive`.
   - Время ответа API сократилось до **3–5 миллисекунд** (для прогретых запросов из кэша — менее **1 миллисекунды**).

2. **Динамическое геокодирование и часовые пояса:**
   - Добавлена полноценная поддержка параметра `city` (и `city1`/`city2` для синастрии) на уровне API.
   - Сервер автоматически запрашивает геоданные и временную зону через сервис `geo_date`.
   - Реализовано автоматическое определение смещения времени (включая летнее/зимнее время) и перевод локального времени в UTC для точных расчетов на швейцарских эфемеридах.

3. **Гибкая фильтрация на стороне сервера:**
   - Внедрена поддержка конфигурации фильтров (флаги `max_orb`, `planets`, `aspects`, `event_types` в URL-запросах).
   - Фильтрация выполняется на уровне Go перед сериализацией данных, уменьшая размер JSON-ответов и нагрузку на сеть.

4. **Исправление ошибок маппинга:**
   - Исправлена критическая ошибка сопоставления ID планет к их строковым именам в событиях календаря.

## Описание страниц сайта

### 1. Главная страница (`index.html`)
* **Файл:** `/index.html`
* **Назначение:** Точка входа на сайт, презентация ключевых возможностей астропроцессора.
* **Описание:** 
  * Содержит приветствие и краткое описание направленности сайта (западная астрология, методы транзитов).
  * Плиточный интерфейс (Grid) для быстрого перехода к основным инструментам: Астрокалендарю, Круговой карте движения планет, Графику транзитов и 3D-карте.

### 2. Астрокалендарь 2D (`calendar2D.html`)
* **Файл:** `/calendar2D.html`
* **Назначение:** Интерактивный лунный и астрологический календарь на месяц.
* **Описание:**
  * Запрашивает данные календаря через API-эндпоинт `/api/v1/calendar`.
  * Позволяет выбрать месяц, год, временную зону, систему зодиака и первый день недели.
  * Три режима представления данных (вкладки):
    * **Таблица:** Интерактивная календарная сетка с возможностью сохранения в виде изображения (📷 через `html2canvas`).
    * **Текст:** Текстовое описание астрологических событий за выбранный период.
    * **График:** Визуальное представление динамики планет.

### 3. Конструктор календаря 2D (`calendar2Dcreate.html`)
* **Файл:** `/calendar2Dcreate.html`
* **Назначение:** Продвинутый инструмент генерации и экспорта астрологического календаря.
* **Описание:**
  * Позволяет гибко задавать временной диапазон (с точностью до минут) и шаг расчетов.
  * Поддерживает экспорт полученных данных в формате JSON или в виде графического файла (изображения).
  * Позволяет анализировать ингрессии (переходы планет в другие знаки) и изменения направления движения (ретроградность).

### 4. Радикс-гороскоп 2D (`radix2D.html`)
* **Файл:** `/radix2D.html`
* **Назначение:** Построение классических плоских (2D) астрологических карт (натальная карта, транзиты, синастрия).
* **Описание:**
  * Интегрирует локальную базу данных карт (`astro-chartsdbview`), позволяя сохранять и загружать профили пользователей.
  * Поддерживает построение различных типов карт: гороскоп, космограмма, транзиты, соляр и лунар.
  * Отрисовка интерактивной карты производится динамически в браузере на основе точных данных, полученных от Go API.

### 5. Радикс-гороскоп 3D (`radix3D.html`)
* **Файл:** `/radix3D.html`
* **Назначение:** Интерактивное трехмерное отображение радикс-карты с привязкой к чакральной системе.
* **Описание:**
  * Построено на технологии WebGL с использованием библиотеки **Three.js**.
  * Позволяет вращать карту в трехмерном пространстве, настраивать глубину, прозрачность и отображение зодиакального круга.
  * Визуализирует соответствие планет чакрам человека (используются 3D-модели чакр из каталога `/models/`, такие как `muladhara.stl`, `anahata.stl` и т.д.).
  * Настройка параметров отображения реализована через интерфейс `lil-gui`.

### 6. Карта-период 2D (`pie2D.html`)
* **Файл:** `/pie2D.html`
* **Назначение:** Круговая интерактивная диаграмма движения планет за выбранный период времени.
* **Описание:**
  * Позволяет отслеживать траектории планет по зодиакальному кругу в динамике.
  * Настраивается стартовая дата, конечная дата, шаг расчета и система координат (топоцентрическая или гелиоцентрическая).
  * Сохраняет пользовательские настройки в `localStorage` для удобства повторного использования.

### 7. График-период 2D (`rect2D.html`)
* **Файл:** `/rect2D.html`
* **Назначение:** Линейный график транзитов планет, совмещенный с внешними экономическими показателями.
* **Описание:**
  * Строит временные графики движения планет и их аспектов.
  * **Уникальная интеграция:** Позволяет накладывать на астрологические графики курсы металлов и валют, загружаемые напрямую с сайта Центробанка РФ (**CBR.RU**).
  * Поддерживает импорт пользовательских данных из файлов формата `.csv` (с помощью библиотеки `PapaParse`), что позволяет сопоставлять любые персональные или рыночные показатели с движением планет.

### 8. Статистика курсов (`course.html`)
* **Файл:** `/course.html`
* **Назначение:** Аналитический инструмент для исследования корреляции между движением планет и финансовыми рынками.
* **Описание:**
  * Позволяет анализировать курсы драгметаллов (Au, Ag, Pt, Pd), фиатных валют (USD, EUR, JPY, CNY) и криптовалюты (BTC) в зависимости от положения планет в градусах, знаках зодиака, а также по дням недели, числам месяца и месяцам года.
  * Строит интерактивные аналитические диаграммы с помощью библиотеки **Plotly.js** (с поддержкой русской локализации).
  * Помогает выявлять статистические закономерности и экстремумы (минимумы/максимумы показателей).

### 9. Эфемериды 2D (`ephemeris2D.html`)
* **Файл:** `/ephemeris2D.html`
* **Назначение:** Генератор классических таблиц эфемерид (точных координат небесных тел).
* **Описание:**
  * Рассчитывает долготу планет на любой заданный шаг времени (от 1 минуты до полугода).
  * Позволяет выводить звездное время (SideTime) и скорость движения планет (в градусах в сутки) для определения ретроградности и стационарности.
  * Поддерживает геоцентрический (Topo) и гелиоцентрический (Gelio) режимы расчетов.

### 10. Настройки (`setting.html`)
* **Файл:** `/setting.html`
* **Назначение:** Единый центр конфигурации астропроцессора, открывающийся в модальном окне.
* **Описание:**
  * Позволяет задать глобальные предпочтения пользователя: направление зодиакального круга (по часовой или против часовой стрелки), точку отсчета (поворот Asc), используемые по умолчанию планеты и аспекты, а также систему домов.
  * Содержит автозаполнение городов (`cities-autocomplete`) и интерактивную карту **Leaflet** для точного определения географических координат (широты/долготы) и часового пояса.
  * Сохраняет все параметры в `localStorage` браузера.

### 11. Обо мне и контакты (`about.html`)
* **Файл:** `/about.html`
* **Назначение:** Персональная страница автора и разработчика сайта.
* **Описание:**
  * Содержит краткую информацию о создателе (Муратов Дамир Юсифович: веб-программист, астролог, психолог, предприниматель) и его натальные данные.
  * Предоставляет ссылки на социальные сети и мессенджеры (Telegram, WhatsApp, Viber, GitHub).

### 12. Политика конфиденциальности (`privacy.html`)
* **Файл:** `/privacy.html`
* **Назначение:** Юридический документ, регламентирующий обработку данных пользователей.
* **Описание:**
  * Декларирует отсутствие сбора персональных данных в явном виде.
  * Поясняет принципы работы с обезличенными данными, собираемыми автоматически через системы аналитики Google Analytics и Яндекс.Метрика.

### 13. Служебная страница (`other.html`)
* **Файл:** `/other.html`
* **Назначение:** Вспомогательная страница для тестирования компонентов и быстрого доступа к ресурсам.
* **Описание:**
  * Содержит таблицы редких астрологических и алхимических символов юникода для копирования.
  * Предоставляет быстрый доступ к тестированию базы данных карт, отображению фаз Луны и подключению различных JS-скриптов.

## Используемые технологии и библиотеки
* **Backend:** Go (Standard Library `net/http` + прямая интеграция расчетных пакетов `astro` и `interactive`).
* **Frontend:** Чистый HTML5, CSS3 (модульная структура фрагментов), современный JavaScript (ES6+ модули).
* **3D-графика:** Three.js (WebGL) для рендеринга трехмерной натальной карты.
* **Диаграммы и графики:** Plotly.js для построения финансово-астрологической статистики.
* **Картография:** Leaflet.js для выбора городов на интерактивной карте.
* **Парсинг данных:** PapaParse.js для обработки импортируемых CSV-файлов.
* **Экспорт:** html2canvas.min.js для создания скриншотов сгенерированных таблиц календаря.
